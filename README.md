# BLAPE Website (multi-page)

## Files
- `index.html` — Home
- `companies.html` — Outreach & Deliverability (Launch / Growth / Scale)
- `individuals.html` — Personal Branding (Profile Fix / Full Presence / Executive)
- `courses.html` — Courses
- `resources.html` — Free resources / lead magnets
